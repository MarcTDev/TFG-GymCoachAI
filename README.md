# GymCoachAI

Bienvenido a **GymCoachAI**, una innovativa aplicación de entrenamiento personalizado que utiliza inteligencia artificial para ayudarte a alcanzar tus objetivos de fitness. Con GymCoachAI, obtendrás rutinas de ejercicio y planes nutricionales adaptados a tus necesidades específicas, todo en una interfaz intuitiva y fácil de usar.

## 🔍 Visión General

GymCoachAI es una aplicación móvil construida con React Native y Expo que combina tecnología moderna con principios de entrenamiento y nutrición para ofrecer una experiencia de fitness personalizada. La aplicación se centra en proporcionar recomendaciones inteligentes mediante el análisis del usuario para generar rutinas efectivas adaptadas a sus objetivos personales.

Principales características:
- **Autenticación segura** con Supabase
- **Rutinas de entrenamiento personalizadas** generadas por IA
- **Planes nutricionales adaptados** a tus necesidades
- **Seguimiento de progreso** con estadísticas detalladas
- **Diseño moderno** con experiencia de usuario optimizada
- **Dashboard interactivo** para control de tu rutina

## 🛠️ Tecnologías Utilizadas

- **Frontend**: React Native con Expo Router
- **Autenticación**: Supabase Auth
- **Base de datos**: Supabase PostgreSQL
- **Estado global**: Context API de React
- **Navegación**: Expo Router con tipos automáticos
- **Estilos**: StyleSheet de React Native
- **Pruebas**: Jest y Testing Library

## 🏃 Uso

1. Regístrate o inicia sesión en la aplicación
2. Completa tu perfil con tus objetivos de fitness
3. Recibe tus rutinas de entrenamiento personalizadas
4. Visualiza tu plan nutricional adaptado
5. Sigue tu progreso en el dashboard
